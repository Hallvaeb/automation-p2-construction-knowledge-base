Git commands:

I den mappen du vil legge git repoet lokalt (den blir lagt til som en ny mappe) skriver du:
git clone https://github.com/Hallvaeb/kbe-a2.git

For å gå inn i repoet:
cd kbe_a2

Så skal alt fungere som før med følgende for å legge til og pushe filer:
git add .
git commit -m "commit-message"
git push
